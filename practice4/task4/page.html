FAQ

Що таке HTML?

HyperText Markup Language — мова розмітки гіпертексту. Описує структуру веб-сторінки.

Чим CSS відрізняється від HTML?

HTML — про структуру. CSS — про оформлення. JavaScript — про поведінку.

Чому JavaScript — це не Java?

Це історична плутанина 1995 року. Це різні мови, спільне лише ім'я.

Що таке DOM?

Document Object Model — програмне представлення HTML як дерева об'єктів. Через нього JavaScript бачить і змінює сторінку.
